# 📌 RPG Escape Dungeon

RPG Escape Dungeon è un gioco di ruolo single-player sviluppato in Java e JavaFX.

Il giocatore esplora un dungeon generato proceduralmente, affronta nemici, raccoglie indizi sulla posizione dell'uscita e cerca di fuggire prima di essere sconfitto.

---

## 🚀 Come eseguire il progetto

### Prerequisiti

* Java 25 (LTS)
* Gradle

### Istruzioni

```bash
git clone <url-del-repository>
cd RPGEscapeDungeon
```

### Build del progetto

```bash
./gradlew build
```

### Esecuzione

```bash
./gradlew run
```

## 🤖 Uso di strumenti di AI

Durante lo sviluppo del progetto sono stati utilizzati strumenti di AI (principalmente ChatGPT) come supporto per:

* chiarimenti teorici su Java, JavaFX e principi SOLID;
* revisione dell'architettura software;
* suggerimenti di refactoring;
* miglioramento della documentazione Javadoc;
* individuazione di possibili miglioramenti progettuali.

Tutte le decisioni progettuali, l'implementazione del codice, l'integrazione delle componenti, il testing e la validazione finale sono stati svolti personalmente.

L'AI è stata utilizzata come strumento di supporto e consulenza tecnica, non come sostituto del processo di sviluppo.
